The small preview images (front and bottom view) allows a simple navigation. The scene can be moved up/down/left/right with left mouse button in the front view. The scene can be moved forward/back with left mouse button in the bottom view. Rotation is done with right mouse button. The scroll wheel can be used to zoom in or zoom out.
The same moves are available in the navigation tab. Here all camera parameters are defined.
